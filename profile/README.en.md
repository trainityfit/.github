## What is Strev?

Freelance personal trainers managing 5–30 clients run their business with **Excel + WhatsApp + handwritten notes**. Strev replaces that stack with a single platform built around the trainer's actual workflow.

> *Training software should not get in the way during a real session.*

<br />

<div align="center">

|  | 🏋️ **Personal trainer** | 🎯 **Independent athlete** |
|---|---|---|
| **Core** | Build routines, assign and customise per client without duplication | Log sessions in 1–2 taps with rest timer and PRs |
| **Management** | Dashboard with KPIs, adherence alerts and stagnation signals | Progress metrics, monthly exportable report |
| **Payments** | Charge Pro/Master/Elite subscriptions via Stripe | Athlete Pro with full history + unlimited AI |
| **Extra** | Digital anamnesis, public trainer profile, communication templates | AI video technique analysis |

</div>
