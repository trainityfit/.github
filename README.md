# Trainity `.github`

This repository exists to power Trainity's public GitHub organization profile and related community-facing metadata.

## What lives here

- `profile/README.md` for the public organization profile
- public-facing defaults and lightweight community metadata, when needed

## Why this repository is public

GitHub requires a public `.github` repository to display a public organization profile README.

## What does not live here

- application source code
- private product documentation
- internal roadmaps
- sensitive operational material

## Internal-only content

If Trainity needs a member-only organization profile, GitHub supports a separate private repository called `.github-private`.
